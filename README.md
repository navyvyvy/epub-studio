# 📚 EPUB STUDIO

브라우저에서 즉시 실행되는 고성능 텍스트-EPUB 배치 변환기입니다.

## ✨ 주요 기능

- **광범위한 기기 호환성 (EPUB 2 & 3)**:
  - 최신 표준인 **EPUB 3(nav.xhtml)**와 구형 e-잉크 단말용 **EPUB 2(toc.ncx)**를 동시 생성하여 리디북스, 킨들, 아이패드 등 모든 기기에서 완벽한 목차를 보장합니다.
- **인터랙티브 드래그 앤 드롭**:
  - 파일을 끌어다 놓기만 하면 즉시 목록에 추가되며, 드래그 시 시각적 피드백을 통해 직관적인 작업이 가능합니다.
- **지능형 목차 추출 엔진**:
  - 복잡한 통합 정규식을 통해 꺽쇠(`<>`), 화괄호(`【】`), 제~화, Chapter 등 다양한 포맷을 자동으로 감지합니다.
- **배치 프로세싱 & 고성능**:
  - 수십 개의 파일을 한 번에 변환하고 ZIP으로 압축 다운로드하며, 모든 처리는 로컬(Zero-Server)에서 이루어져 안전합니다.
- **스튜디오 테마 UI**:
  - 장시간 작업에도 눈이 편안한 다크 모드 기반의 전문적인 작업 환경을 제공합니다.

## ⚙️ 목차 인식 정규식 가이드 (최신화)

실제 사용 중인 통합 패턴에 맞춰 최적화된 가이드를 제공합니다.

| 패턴 이름 | 설명 및 인식 대상 |
| :--- | :--- |
| **통합 패턴** | 괄호(`【】`, `< >`), 제~화, Chapter, #숫자 등을 모두 포함하는 강력한 패턴 |
| **특수 괄호** | `【001】`, `<공지>`, `〔제1화〕` 등 특수 기호로 감싸진 제목 |
| **플랫폼 표준** | `제 10화. 본문 시작` (카카오페이지, 문피아 등 주요 플랫폼 형식) |
| **영문/넘버링** | `Chapter 1`, `#123`, `(1)` 등 영문 및 단순 숫자 넘버링 |
| **일반 제목** | `1. 새로운 시작` 등 숫자로 시작하며 특정 종결어미가 없는 문장 |

## 🚀 시작하기

1. 변환할 텍스트 파일을 중앙의 **Drop Zone**으로 드래그합니다.
2. 우측의 **정규식 가이드**에서 본문에 맞는 샘플을 클릭하거나 직접 패턴을 입력합니다.
3. 변환 버튼을 눌러 생성된 EPUB 또는 ZIP 파일을 다운로드합니다.
