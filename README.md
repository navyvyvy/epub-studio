# 📚 EPUB STUDIO

브라우저에서 즉시 실행되는 고성능 텍스트-EPUB 배치 변환기입니다.

## ✨ 주요 기능

- **광범위한 기기 호환성 (EPUB 2 & 3)**:
  - 최신 표준인 **EPUB 3(nav.xhtml)**와 구형 e-잉크 단말기에서 사용하는 **EPUB 2(toc.ncx)** 목차 형식을 모두 생성합니다.
  - 리디북스, 킨들, 구글 북스, 아이패드 등 모든 환경에서 목차가 완벽하게 작동합니다.
- **지능형 목차 추출**: 정규식(Regex)을 사용하여 텍스트 내의 장/절을 자동으로 감지하고 목차를 생성합니다.
- **다양한 인코딩 지원**: `jschardet`를 통해 UTF-8, CP949(EUC-KR) 등 인코딩을 자동 판별하여 글자 깨짐을 방지합니다.
- **배치 프로세싱**: 수십 개의 파일을 한 번에 변환하고 ZIP 압축 다운로드를 지원합니다.
- **Zero-Server**: 모든 작업은 로컬 브라우저 내에서 처리되어 개인정보가 외부로 유출되지 않습니다.

## ⚙️ 목차 인식 정규식 가이드

이 도구는 정규식을 통해 목차를 구분합니다. 본문에 맞는 패턴을 선택하세요.

| 패턴 이름 | 정규식 예시 | 인식 대상 |
| :--- | :--- | :--- |
| **기본** | `^\s*(제?\s*\d+\s*[장화권]...` | 제 1장, 제 2화, Chapter 5 등 |
| **숫자형** | `^\s*\d+[\.\s]` | 1. 시작, 2. 전개 (숫자로 시작하는 문장) |
| **해시태그** | `^\s*#\d+` | #1, #2... |
